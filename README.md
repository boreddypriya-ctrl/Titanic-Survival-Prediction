# Titanic Survival Prediction

## Intern Details

**Intern ID:** CITS4322

## Project Overview

This machine learning project predicts whether a passenger survived the Titanic disaster using passenger information such as age, gender, passenger class, fare, and family details.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn

## Dataset

Titanic Dataset from Kaggle.

## Features Used

* Passenger Class (Pclass)
* Gender (Sex)
* Age
* Siblings/Spouses (SibSp)
* Parents/Children (Parch)
* Fare

## Machine Learning Model

Random Forest Classifier

## Results

Model Accuracy: 81.01%

## How to Run

1. Install required libraries:
   pip install pandas numpy scikit-learn matplotlib

2. Run the project:
   py titanic_survival_prediction.py
